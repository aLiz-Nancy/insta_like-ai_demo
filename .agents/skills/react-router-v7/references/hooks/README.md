# React Router v7 — Hooks

| Name | Description | Path |
|------|-------------|------|
| `useLoaderData` | Returns data from the closest route's `loader` or `clientLoader` | [./useLoaderData.md](./useLoaderData.md) |
| `useActionData` | Returns data from the route's `action` after a form submission | [./useActionData.md](./useActionData.md) |
| `useNavigation` | Returns the current `Navigation` object for pending UI | [./useNavigation.md](./useNavigation.md) |
| `useNavigate` | Returns a function for programmatic navigation | [./useNavigate.md](./useNavigate.md) |
| `useFetcher` | Creates a fetcher for loading/submitting without navigation | [./useFetcher.md](./useFetcher.md) |
| `useFetchers` | Returns all in-flight fetchers for optimistic UI | [./useFetchers.md](./useFetchers.md) |
| `useParams` | Returns dynamic route parameters from the current URL | [./useParams.md](./useParams.md) |
| `useRouteLoaderData` | Returns loader data for any route by its route ID | [./useRouteLoaderData.md](./useRouteLoaderData.md) |
| `useMatches` | Returns all active route matches in the current hierarchy | [./useMatches.md](./useMatches.md) |
| `useLocation` | Returns the current `Location` object | [./useLocation.md](./useLocation.md) |
| `useSubmit` | Returns a function to submit forms programmatically | [./useSubmit.md](./useSubmit.md) |
| `useRouteError` | Returns the error thrown in a route's loader, action, or render | [./useRouteError.md](./useRouteError.md) |
| `useRevalidator` | Returns a function to manually revalidate route loader data | [./useRevalidator.md](./useRevalidator.md) |
| `useSearchParams` | Returns current URL search params and a setter function | [./useSearchParams.md](./useSearchParams.md) |
| `useHref` | Resolves a path to a fully-qualified href string | [./useHref.md](./useHref.md) |
| `useBlocker` | Blocks in-app navigations and exposes a confirmation UI API | [./useBlocker.md](./useBlocker.md) |
| `useBeforeUnload` | Registers a callback on the browser's `beforeunload` event | [./useBeforeUnload.md](./useBeforeUnload.md) |
| `useAsyncValue` | Returns the resolved value from the nearest `<Await>` component | [./useAsyncValue.md](./useAsyncValue.md) |
| `useAsyncError` | Returns the rejection error from the nearest `<Await>` component | [./useAsyncError.md](./useAsyncError.md) |
| `useOutletContext` | Returns context passed via `<Outlet context={...}>` | [./useOutletContext.md](./useOutletContext.md) |
| `useViewTransitionState` | Returns `true` when a View Transition to a location is active | [./useViewTransitionState.md](./useViewTransitionState.md) |
| `useFormAction` | Resolves the closest route URL, optionally with a sub-action path | [./useFormAction.md](./useFormAction.md) |
| `useMatch` | Returns a `PathMatch` if a pattern matches the current URL, else `null` | [./useMatch.md](./useMatch.md) |
