# Configuration

syncpack configuration options reference.

| Name | Description | Path |
|------|-------------|------|
| syncpackrc | .syncpackrc config file format and search order | [./syncpackrc.md](./syncpackrc.md) |
| customTypes | Define custom package.json properties to manage beyond standard dependency types | [./custom-types.md](./custom-types.md) |
| dependencyGroups | Group multiple dependencies to be treated as a single entity for version management | [./dependency-groups.md](./dependency-groups.md) |
| formatBugs | Format the `bugs` field of package.json into shorthand format | [./format-bugs.md](./format-bugs.md) |
| formatRepository | Format the `repository` field of package.json into shorthand format | [./format-repository.md](./format-repository.md) |
| indent | Characters used when writing formatted JSON to package.json files | [./indent.md](./indent.md) |
| maxConcurrentRequests | Maximum number of concurrent HTTP requests to the npm registry | [./max-concurrent-requests.md](./max-concurrent-requests.md) |
| semverGroups | Allow dependencies to have different semver range rules per group | [./semver-groups.md](./semver-groups.md) |
| sortAz | Alphabetically sort specified object keys or array values in package.json | [./sort-az.md](./sort-az.md) |
| sortExports | Order the `exports` field properties in package.json | [./sort-exports.md](./sort-exports.md) |
| sortFirst | Specify which package.json fields appear at the top | [./sort-first.md](./sort-first.md) |
| sortPackages | Alphabetically sort root-level properties of package.json | [./sort-packages.md](./sort-packages.md) |
| source | Glob patterns to discover package.json files | [./source.md](./source.md) |
| strict | Exit with an error when version mismatches are detected | [./strict.md](./strict.md) |
