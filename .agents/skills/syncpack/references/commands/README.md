# Commands

syncpack CLI コマンド一覧。

| Name | Description | Path |
|------|-------------|------|
| fix | Autofix dependency version mismatches across a monorepo to match configured rules | [./fix.md](./fix.md) |
| format | Sort `package.json` fields into consistent order and alphabetize nested fields | [./format.md](./format.md) |
| json | Export all dependency instances as newline-delimited JSON for use with `jq` or other tools | [./json.md](./json.md) |
| lint | Check for dependency version mismatches; exits with code `1` if issues found | [./lint.md](./lint.md) |
| list | Display all dependencies with versions, locations, and mismatch status | [./list.md](./list.md) |
| update | Update dependencies to newer versions from the npm registry | [./update.md](./update.md) |
