# Testing

Pino provides a dedicated utility for testing log output.

## pino-test

Use [`pino-test`](https://github.com/pinojs/pino-test) for verifying logs generated by the Pino logger in your test suites.

## Notes

- `pino-test` is the officially recommended utility for testing Pino log output
- It is part of the core Pino ecosystem and maintained by the Pino team
- Use it to assert on log levels, messages, and structured data in tests
