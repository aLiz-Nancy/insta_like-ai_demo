# mise でのインストール

Source: https://lefthook.dev/installation/mise

## インストールコマンド

```
mise use lefthook@latest
```

Mise バージョンマネージャーを使用して Lefthook をインストールします。プロジェクト内でツール依存として管理することが可能です。

## 注意事項

Lefthook の mise プラグインはコミュニティによってメンテナンスされています。Lefthook チームは mise 固有のインストール問題について直接的なサポートを提供できない場合があります。
