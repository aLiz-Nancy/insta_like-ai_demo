# React Flow — Hooks

| Name | Description | Path |
|------|-------------|------|
| `useConnection` | 進行中の接続操作の状態を取得 | [./useConnection.md](./useConnection.md) |
| `useEdges` | 現在のエッジ配列を取得 | [./useEdges.md](./useEdges.md) |
| `useEdgesState` | エッジの状態と変更ハンドラを管理（非推奨） | [./useEdgesState.md](./useEdgesState.md) |
| `useHandleConnections` | 特定ハンドルの接続情報を取得 | [./useHandleConnections.md](./useHandleConnections.md) |
| `useInternalNode` | 内部ノードオブジェクト（寸法・位置含む）を取得 | [./useInternalNode.md](./useInternalNode.md) |
| `useKeyPress` | 特定キーの押下状態を監視 | [./useKeyPress.md](./useKeyPress.md) |
| `useNodeConnections` | 特定ノードの全接続情報を取得 | [./useNodeConnections.md](./useNodeConnections.md) |
| `useNodeId` | カスタムノード内で自身のノード ID を取得 | [./useNodeId.md](./useNodeId.md) |
| `useNodes` | 現在のノード配列を取得 | [./useNodes.md](./useNodes.md) |
| `useNodesData` | 指定ノードの data プロパティを取得 | [./useNodesData.md](./useNodesData.md) |
| `useNodesInitialized` | 全ノードの寸法計算完了を検知 | [./useNodesInitialized.md](./useNodesInitialized.md) |
| `useNodesState` | ノードの状態と変更ハンドラを管理（非推奨） | [./useNodesState.md](./useNodesState.md) |
| `useOnSelectionChange` | ノード/エッジの選択状態変更を監視 | [./useOnSelectionChange.md](./useOnSelectionChange.md) |
| `useOnViewportChange` | ビューポートの変更（パン・ズーム）を監視 | [./useOnViewportChange.md](./useOnViewportChange.md) |
| `useReactFlow` | React Flow インスタンスのメソッドにアクセス | [./useReactFlow.md](./useReactFlow.md) |
| `useStore` | React Flow の内部 Zustand ストアから値を取得 | [./useStore.md](./useStore.md) |
| `useStoreApi` | React Flow の内部 Zustand ストア API に直接アクセス | [./useStoreApi.md](./useStoreApi.md) |
| `useUpdateNodeInternals` | ノードの内部状態（ハンドル位置等）を手動更新 | [./useUpdateNodeInternals.md](./useUpdateNodeInternals.md) |
| `useViewport` | 現在のビューポート（x, y, zoom）を取得 | [./useViewport.md](./useViewport.md) |
