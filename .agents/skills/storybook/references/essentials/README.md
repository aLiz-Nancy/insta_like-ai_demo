# Storybook Essentials

Core addons bundled with Storybook that cover the most common development and debugging needs.

| Name | Description | Path |
|------|-------------|------|
| Actions | Spy on event handler args and display them in the actions panel | [./actions.md](./actions.md) |
| Backgrounds | Switch the canvas background color to test components against different schemes | [./backgrounds.md](./backgrounds.md) |
| Controls | Interactively adjust component args from the Controls panel without code changes | [./controls.md](./controls.md) |
| Highlight | Overlay visual highlights on specific DOM nodes for accessibility and layout debugging | [./highlight.md](./highlight.md) |
| Measure & Outline | Inspect element dimensions/spacing (Measure) and visualize alignment (Outline) via toolbar | [./measure-and-outline.md](./measure-and-outline.md) |
| Toolbars & Globals | Add custom toolbar controls that set shared global state consumed by decorators and stories | [./toolbars-and-globals.md](./toolbars-and-globals.md) |
| Viewport | Resize the story iframe to simulate different screen sizes and device resolutions | [./viewport.md](./viewport.md) |
