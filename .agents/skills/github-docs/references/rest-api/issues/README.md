# Issues API

GitHub Issues API のリファレンスセクション。

| ファイル | 説明 |
|---|---|
| [issues.md](./issues.md) | Issue の CRUD 操作（一覧取得・作成・更新・ロック/アンロック） |
| [comments.md](./comments.md) | Issue コメントの一覧取得・作成・更新・削除・ピン留め |
| [labels.md](./labels.md) | ラベルの CRUD 操作（リポジトリラベル・Issue ラベル） |
| [milestones.md](./milestones.md) | マイルストーンの一覧取得・作成・更新・削除 |
| [assignees.md](./assignees.md) | アサイニーの確認・追加・削除 |
