# Organizations API

GitHub Organizationの情報を取得・管理するAPIセクション。

## エンドポイントカテゴリ

| カテゴリ | 説明 |
|---------|------|
| [Organizations](orgs.md) | Organization情報の取得・更新・削除、インストール一覧の取得 |

## 概要

Organizations APIは、GitHub Organizationのプロフィール情報の管理、所属Organization一覧の取得、およびOrganizationにインストールされたGitHub Appの一覧取得を提供する。
