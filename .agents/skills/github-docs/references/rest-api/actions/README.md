# Actions API

GitHub Actions API のリファレンスセクション。

| ファイル | 説明 |
|---|---|
| [workflows.md](./workflows.md) | ワークフローの一覧取得・有効化/無効化・手動実行・使用状況 |
| [workflow-runs.md](./workflow-runs.md) | ワークフロー実行の一覧取得・再実行・キャンセル・ログ・デプロイメント承認 |
| [workflow-jobs.md](./workflow-jobs.md) | ワークフロージョブの取得・ジョブ一覧・ログダウンロード |
| [artifacts.md](./artifacts.md) | アーティファクトの一覧取得・ダウンロード・削除 |
| [secrets.md](./secrets.md) | シークレットの CRUD 操作（暗号化必須・値は返却されない） |
| [variables.md](./variables.md) | 変数の CRUD 操作（値が返却される） |
| [self-hosted-runners.md](./self-hosted-runners.md) | セルフホストランナーの管理・トークン発行・ラベル操作 |
