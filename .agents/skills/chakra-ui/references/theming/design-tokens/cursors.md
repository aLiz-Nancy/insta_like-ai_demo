# Cursors

Semantic cursor tokens for interactive UI elements.

## Tokens

| Token | CSS Variable | Value |
|-------|-------------|-------|
| `button` | `--chakra-cursor-button` | `pointer` |
| `checkbox` | `--chakra-cursor-checkbox` | `default` |
| `disabled` | `--chakra-cursor-disabled` | `not-allowed` |
| `menuitem` | `--chakra-cursor-menuitem` | `default` |
| `option` | `--chakra-cursor-option` | `default` |
| `radio` | `--chakra-cursor-radio` | `default` |
| `slider` | `--chakra-cursor-slider` | `default` |
| `switch` | `--chakra-cursor-switch` | `pointer` |

## Notes

- Tokens follow semantic naming conventions tied to component types
- `disabled` token applies `not-allowed` to communicate non-interactivity
- Applied automatically to Chakra UI built-in components
