# Installation

Install Chakra UI v3 and its peer dependencies, then set up the provider for your project.

## Installation

```bash
npm i @chakra-ui/react @emotion/react
```

## Add Snippets

Snippets are pre-built component compositions that speed up development. Use the CLI to add them:

```bash
npx @chakra-ui/cli snippet add
```

## Update tsconfig

For TypeScript projects, update `compilerOptions` in `tsconfig.json`:

```json
{
  "compilerOptions": {
    "target": "ESNext",
    "module": "ESNext",
    "moduleResolution": "Bundler",
    "skipLibCheck": true,
    "paths": {
      "@/*": ["./src/*"]
    }
  }
}
```

For JavaScript projects, create `jsconfig.json` with the same content.

## Setup Provider

Wrap your application root with the `Provider` component from `components/ui/provider` (generated by the snippet CLI). The provider composes:

- `ChakraProvider` from `@chakra-ui/react` — styling system
- `ThemeProvider` from `next-themes` — color mode

## Notes

- Minimum Node.js version: **20.x**
- Chakra UI uses Emotion at runtime; a zero-runtime model (inspired by Panda CSS) is planned for future phases
- Try Chakra UI instantly in a [StackBlitz sandbox](https://stackblitz.com) without local installation
