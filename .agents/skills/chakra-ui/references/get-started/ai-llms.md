# AI LLMs

Chakra UI v3 provides LLMs.txt files that make the documentation accessible to AI coding tools such as Cursor, Windsurf, GitHub Copilot, ChatGPT, and Claude.

## Available Files

| File | URL | Description |
|------|-----|-------------|
| Main | `https://chakra-ui.com/llms.txt` | Primary LLMs.txt file |
| Full | `https://chakra-ui.com/llms-full.txt` | Complete Chakra UI v3 documentation |
| Components | `https://chakra-ui.com/llms-components.txt` | Component documentation only |
| Styling | `https://chakra-ui.com/llms-styling.txt` | Styling documentation only |
| Theming | `https://chakra-ui.com/llms-theming.txt` | Theming documentation only |
| Migration | `https://chakra-ui.com/llms-v3-migration.txt` | Migration guide for upgrading to v3 |

## Usage

### Cursor

Use the `@Docs` feature to include LLMs.txt files in your project context.

See [Cursor @Docs documentation](https://docs.cursor.com/context/@-symbols/@-docs) for full instructions.

### Windsurf

Reference the files using `@` mentions or include them in `.windsurfrules` files.

See [Windsurf memories and rules documentation](https://docs.codeium.com/windsurf/memories#memories-and-rules) for full instructions.

## Notes

- Use `llms-v3-migration.txt` specifically when migrating from v2 — it contains migration-specific guidance
- Use `llms-components.txt` when working primarily with component usage questions
