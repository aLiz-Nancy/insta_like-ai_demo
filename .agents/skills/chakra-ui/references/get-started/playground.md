# Playground

The Chakra UI Playground is an interactive demo environment for exploring Chakra UI components live in the browser.

## Access

URL: [https://chakra-ui.com/playground](https://chakra-ui.com/playground)

## Description

The Playground is a live, interactive interface — not a documentation page. It lets you:

- Preview component variants (Button, Tabs, Avatar, Checkbox, etc.) with different props and accent colors
- Explore the design token system (colors, spacing, typography, animations) visually
- Experiment with Chakra UI components without a local setup

## Notes

- No installation required — runs entirely in the browser
- Useful for quickly prototyping component configurations before writing code
- The Playground reflects the current stable version of Chakra UI
