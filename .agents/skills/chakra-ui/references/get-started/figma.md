# Figma

The official Chakra UI v3 Figma Kit lets designers work with the same building blocks that exist in code, ensuring design decisions translate directly into production components.

## Access

The Chakra UI v3 Figma Kit is freely available on Figma Community — no special permissions or paid subscription required.

**URL:** Search for "Chakra UI v3" on [Figma Community](https://www.figma.com/community)

## Features

- All components mirror Chakra UI's production component structure and behavior
- Design tokens (colors, spacing, typography) match the code-side design system
- Supports consistent collaboration between designers and developers
- Reduces friction in the design-to-development handoff

## Notes

- The kit reflects how Chakra UI actually functions in production, not a visual approximation
- Component updates in the kit track changes in the Chakra UI codebase
