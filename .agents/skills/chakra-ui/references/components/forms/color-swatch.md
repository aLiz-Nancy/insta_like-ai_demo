# Color Swatch

Displays a color swatch box in various sizes and shapes.

## Import

```tsx
import { ColorSwatch } from "@chakra-ui/react"
```

## Usage

```tsx
<ColorSwatch value="#3b82f6" size="md" />
```

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | **required** | The color value to display |
| `size` | `"2xs" \| "xs" \| "sm" \| "md" \| "lg" \| "xl" \| "2xl" \| "full"` | `"md"` | The size of the swatch |
| `shape` | `"square" \| "circle" \| "rounded"` | `"rounded"` | The shape of the swatch |

## Variants / Sizes

- **sizes**: `2xs`, `xs`, `sm`, `md`, `lg`, `xl`, `2xl`, `full`
- **shapes**: `square`, `circle`, `rounded`
