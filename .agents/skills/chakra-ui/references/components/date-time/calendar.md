# Calendar

An inline calendar component for selecting dates or date ranges, displayed without a popover trigger.

## Import

```tsx
import { Calendar } from "@chakra-ui/react"
```

## Usage

```tsx
<Calendar />
```

## Notes

- Renders an inline calendar (no input or popover wrapper)
- Useful when you want a persistent calendar display on the page
- For a date input with a popover calendar, use [`DatePicker`](./date-picker.md) instead
- Accepts the same core date selection props as `DatePicker`
